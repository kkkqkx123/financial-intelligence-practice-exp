## 可视化建议

### 1. 基础仪表板
```cypher
// 关键指标卡片
MATCH (c:公司) RETURN count(c) as 公司数量
MATCH (i:投资方) RETURN count(i) as 投资方数量
MATCH ()-[r:投资]-() RETURN count(r) as 投资关系数
```

### 2. 投资网络图
```cypher
// 投资关系网络
MATCH (i:投资方)-[r:投资]->(c:公司)
RETURN i, r, c
LIMIT 50
```

### 3. 时间序列分析
```cypher
// 投资时间趋势
MATCH (i:投资方)-[r:投资]->(c:公司)
WHERE r.投资时间 IS NOT NULL
RETURN date(r.投资时间).year as 年份, count(*) as 投资次数
ORDER BY 年份
```

### 4. 行业投资热力图
```cypher
// 行业投资分布（需要行业数据）
MATCH (c:公司)-[:属于]->(ind:行业), (i:投资方)-[r:投资]->(c)
RETURN ind.行业名称, count(r) as 投资次数
ORDER BY 投资次数 DESC
```
