#方向：知识图谱与大模型应用

本作业旨在通过大模型技术，自动化生成针对特定金融领域的知识图谱。需要从金融文本中抽取实体和关系，基于大模型实现知识图谱的构建，并且将知识图谱存储在Neo4j数据库中。

- 数据来源：此链接https://github.com/smoothnlp/FinancialDatasets 提供了企业工商信息、金融讯息新闻、专栏资讯、投资机构信息、投资事件、36氪新闻等金融数据。可以根据自己的需要，从中节选小样本数据集进行实验。
- 大模型：可采用公开Huggingface平台https://huggingface.co/上开源的预训练模型，也可以使用GPT、通义千问、文心一言、ChatGLM等模型API进行开发。
