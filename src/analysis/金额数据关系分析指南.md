# 金额数据关系分析指南

## 1. 数据概况与价值分析

### 1.1 现有数据资产
- **投资事件数据**: 3,836条投资关系，包含金额、轮次、时间等关键信息
- **投资方数据**: 3,023个投资方，包含投资偏好和行业专注度
- **公司数据**: 1,647家公司，包含基本信息和融资历史
- **行业分类**: 25个行业，覆盖主要投资领域
- **轮次信息**: 21个投资轮次，反映企业发展阶段

### 1.2 金额数据价值
- **投资规模分析**: 了解不同轮次、行业的投资规模分布
- **投资趋势分析**: 基于时间序列分析投资热度变化
- **风险评估**: 通过金额分布评估投资风险
- **决策支持**: 为投资决策提供数据支撑

## 2. 核心分析维度

### 2.1 金额与轮次关系分析
**分析目标**: 揭示不同轮次的投资规模特征
- **统计指标**: 平均金额、中位数、最大值、最小值、标准差
- **可视化**: 箱线图、柱状图、散点图
- **深度分析**: 轮次间金额差异显著性检验

### 2.2 金额与行业关系分析  
**分析目标**: 识别高价值投资行业
- **行业投资规模排名**: 按总金额、平均金额排序
- **行业投资集中度**: 分析行业内部金额分布
- **趋势分析**: 行业投资规模随时间变化

### 2.3 金额与时间关系分析
**分析目标**: 分析投资市场周期性
- **年度/季度趋势**: 投资金额的时间分布
- **季节性分析**: 特定时期的投资活跃度
- **市场周期**: 识别投资高峰期和低谷期

### 2.4 金额与投资方关系分析
**分析目标**: 分析投资方的投资策略
- **投资方规模分布**: 不同投资方的投资金额特征
- **投资偏好**: 投资方在轮次、行业的金额分配
- **投资策略**: 分析投资方的风险偏好

## 4. Neo4j图数据库应用

### 4.1 数据模型设计
```cypher
// 金额统计节点
CREATE (s:金额统计 {统计类型: '轮次金额分布', 轮次: 'A轮', 平均金额: 50000000, 中位数: 30000000})

// 金额与轮次关系
MATCH (r:轮次 {轮次名称: 'A轮'}), (s:金额统计)
CREATE (r)-[:金额分布 {平均金额: 50000000, 投资次数: 100}]->(s)

// 金额与行业关系  
MATCH (i:行业 {行业名称: '企业服务'}), (s:金额统计)
CREATE (i)-[:行业投资规模 {总金额: 1000000000, 平均金额: 80000000}]->(s)
```

### 4.2 查询模式设计

#### 4.2.1 基础查询
```cypher
// 查询各轮次的金额统计
MATCH (r:轮次)-[rel:金额分布]->(s:金额统计)
RETURN r.轮次名称, rel.平均金额, rel.投资次数
ORDER BY rel.平均金额 DESC

// 查询各行业的投资规模
MATCH (i:行业)-[rel:行业投资规模]->(s:金额统计)
RETURN i.行业名称, rel.总金额, rel.平均金额
ORDER BY rel.总金额 DESC
```

#### 4.2.2 高级分析查询
```cypher
// 投资方投资规模分析
MATCH (investor:投资方)-[r:投资]->(c:公司)
WHERE r.金额 IS NOT NULL
WITH investor, COUNT(r) as 投资次数, AVG(TOINTEGER(r.金额)) as 平均金额, SUM(TOINTEGER(r.金额)) as 总投资额
RETURN investor.投资方名称, 投资次数, 平均金额, 总投资额
ORDER BY 总投资额 DESC
LIMIT 20

// 时间序列金额分析
MATCH ()-[r:投资]->()
WHERE r.金额 IS NOT NULL AND r.融资时间 IS NOT NULL
WITH substring(r.融资时间, 0, 4) as 年份, AVG(TOINTEGER(r.金额)) as 年均金额, COUNT(r) as 年投资次数
RETURN 年份, 年均金额, 年投资次数
ORDER BY 年份
```
