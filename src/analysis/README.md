# 知识图谱分析报告目录

## 报告概览
本目录包含基于金融投资知识图谱的全面分析报告，涵盖投资方、行业、网络、轮次和综合分析五个维度。

## 报告列表

### 1. 投资方分析报告
- **文件**: `投资方分析报告.md`
- **内容**: 投资方活跃度、轮次偏好、行业覆盖分析
- **核心发现**: 投资方活跃度分布、轮次偏好特征、行业覆盖广度

### 2. 行业分析报告  
- **文件**: `行业分析报告.md`
- **内容**: 热门行业投资分析、投资方覆盖广度、投资活跃度
- **核心发现**: 企业服务、医疗、文娱为投资热点行业

### 3. 网络分析报告
- **文件**: `网络分析报告.md`
- **内容**: 投资关系网络结构、连接度分布、投资强度分析
- **核心发现**: 稀疏网络特征、核心-边缘结构、1:1关系模式

### 4. 轮次分析报告
- **文件**: `轮次分析报告.md`
- **内容**: 投资轮次偏好分布、投资方覆盖广度、轮次阶段分析
- **核心发现**: 成长期投资主导、早期投资活跃、轮次覆盖完整

### 5. 综合分析报告
- **文件**: `综合分析报告.md`
- **内容**: 整合四个维度的分析结果，提供整体投资生态洞察
- **核心发现**: 投资生态结构特征、投资策略类型、市场趋势判断

## 数据基础
- **知识图谱规模**: 2,219个节点，1,388个关系
- **数据来源**: 投资事件数据、投资结构数据
- **数据库**: Neo4j图数据库

## 分析方法
- **统计分析**: 频次分布、排名分析
- **网络分析**: 连接度、中心性、网络结构
- **交叉分析**: 行业-轮次、投资方-行业关系

## 主要发现摘要

### 投资热点
- **热门行业**: 企业服务(106)、医疗(68)、文娱内容游戏(65)
- **主流轮次**: A轮(118)、B轮(103)、Angel轮(91)

### 网络特征
- **稀疏网络**: 投资方-行业关系呈现稀疏性
- **1:1模式**: 每个投资方在每个行业/轮次平均只有1次关系
- **核心节点**: 少数行业和投资方占据网络核心位置

### 投资策略
- **专业化**: 47个投资方仅覆盖1个行业
- **多元化**: 1个投资方覆盖25个行业
- **阶段专注**: 成长期投资占比46.4%

## 使用说明
1. 各报告可独立阅读，了解特定维度的分析结果
2. 综合分析报告提供整体视角和洞察
3. 报告包含具体数据、图表和分析结论
4. 所有分析基于真实知识图谱数据
