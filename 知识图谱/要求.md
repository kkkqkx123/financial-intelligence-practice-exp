  ```python
  uri = "bolt://localhost:7687"
  ```

- c. 创建实体：基于 py2neo.Node 依次创建企业、行业、股东、概念实体，并且补充相关节点属性，请避免实体重复创建问题。
- d. 创建关系：基于 py2neo.Relationship 依次创建企业-企业、企业-行业、企业-股东、企业-概念关系，并且补充相关关系属性。
  - i. 基于 py2neo.NodeMatcher 查找 node
  - ii. 基于企业的股票之间的皮尔逊相关系数来创建企业-企业关系。皮尔逊相关系数的计算代码如下所示。
    ```python
    import numpy as np
    def compute_corrcoef(stock_return: np.ndarray):
        corrcoef = np.corrcoef(stock_return)
        corrcoef[np.isnan(corrcoef)] = 0  # nan位置置0
        return corrcoef
    ```
