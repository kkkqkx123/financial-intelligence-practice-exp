# 知识图谱与大模型应用

本项目旨在通过大模型技术，自动化生成针对特定金融领域的知识图谱。
需要从金融文本中抽取实体和关系，基于大模型实现知识图谱的构建，并且将知识图谱存储在Neo4j数据库中。

数据集为src\dataset\company_data.csv
